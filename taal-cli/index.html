Taal CLI Downloads

The Taal CLI is designed to allow quick and easy implementation of all Stas token features from the command line. We can mint tokens, transfer, redeem, merge etc, all without any technical understanding of the underlying protocol or programming knowlege. The CLI can be used to expirement with the various sdk functions before building your application.

Please read the TAAL licence carefully before using this software.

Instructions

taal-cli does not need a special installation. You can copy the executable to a directory and launch from there..

The macOS requires permissions to be set in the terminal 'chmod 755 taal-cli'
