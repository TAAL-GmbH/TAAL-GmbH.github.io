Source code

The source code can be found in https://github.com/TAAL-GmbH/taal-client

Instructions

1. You will need to create a free account with Taal at https://console.taal.com/register

2. Log in and then create a new API Key in the KEYS section.

3. Register the API key with your instance of Taal Client (taal-client register )

4. Start the Taal Client (taal-client start)

Usage
-----
taal-client register 
  Creates a private key which is held locally, and sends the public key to Taal to be linked to an existing API key.

taal-client start
  Starts listening for requests on :9500.  This value can be changed with the LISTEN environment variable.

  All requests will be sent to https://tapi.taal.com by default unless overridden with the TAAL_URL environment variable.

  DEBUG = 1 will log all transactions to console

Environment variables
---------------------
  LISTEN
  TAAL_URL
  TAAL_TIMEOUT
  DEBUG

Example
-------
  DEBUG=1 LISTEN=localhost:8080 TAAL_URL=http://localhost:4000 TAAL_TIMEOUT=1m ./taal_client start

API keys

To use the Taal Client you will need a valid API key. This can be obtained from https://console.taal.com

cURL example

To write data, you fire a POST request to http://localhost:9500/api/v1/write. The payload of the POST will be written to the blockchain along with the MIME type that's provided in the Content-Type header.

Optionally, there are 2 additional headers that can be set:
1. X-Tag - the value of this header will be prefixed to the MIME type and payload in the transaction.
2. X-FeesRequired - if this header exists and it is set to "true", Taal' servers will construct transactions with additional fees for the miners. 

curl \
--location \
--request POST 'http://localhost:9500/api/v1/write' \
--header 'X-FeesRequired: true' \
--header 'Authorization: Bearer testnet_00000000000000000000000000000000' \
--header 'Content-Type: text/plain' \
--data-raw 'Hello world!'
An example transaction can be seen at https://test.whatsonchain.com/tx/659ea82b9abcbfdbaaf168b24ad164f0be7e3b50e9c51dcff275a8d5a44e12ed
